# Driver-BezierCurvePath

cocos creator 2.4.0 写的一个简单的曲线运动demo
```
Author:JerryYang 
creator by 2020.07.10 
```


- 利用贝塞尔曲线让物体做曲线运动，并且把曲线绘制出来;
- 可以实时改变曲线的样式（可视化编辑曲线）;
- 控制物体沿曲线的的正方向。
